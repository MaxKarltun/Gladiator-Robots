## ⚔️ About the Game

**Gladiator Robots** is a browser-based JavaScript game where you battle enemy robots in a turn-based arena.

You’ll make strategic choices each round:
- Fight or skip
- Spend money in the shop
- Upgrade health and attack
- Try to finish with the highest score

It’s lightweight, fun, and a great project for learning JavaScript fundamentals like functions, loops, conditionals, and object manipulation.

---

## ✨ Features

- Turn-based robot combat
- Player choices using prompt/alert interactions
- In-game shop system (health and attack upgrades)
- Score and currency tracking
- Randomized enemy stats for replayability

## 🚀 Run Locally

Because this project is plain HTML + JavaScript, no build tools are required.

### Option 1: Open directly in browser
1. Clone this repository:
   ```bash
   git clone https://github.com/MaxKarltun/Gladiator-Robots.git
   ```
2. Open the project folder.
3. Double-click `index.html` to launch the game in your browser.

### Option 2: Use VS Code Live Server (recommended)
1. Open the repo in VS Code.
2. Install the **Live Server** extension.
3. Right-click `index.html` → **Open with Live Server**.

## 🌐 Deployment

You can deploy this game for free with **GitHub Pages**.

### Deploy with GitHub Pages
1. Push your latest code to GitHub.
2. Go to your repository:  
   `Settings` → `Pages`
3. Under **Build and deployment**:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (or your default branch)
   - **Folder**: `/ (root)`
4. Click **Save**.
5. Wait ~1–2 minutes, then open your live site URL shown on the Pages settings screen.

## 🕹️ How to Play

1. Enter your robot’s name.
2. Battle enemy robots one round at a time.
3. Choose to **fight** or **skip** each turn.
4. Earn money from wins.
5. Visit the shop between rounds to upgrade your robot.
6. Defeat all enemies and set a high score.
